Propiedades de HTML Explicadas

CSS

En esta página se describen las principales propiedades de HTML y sus usos.

1. Propiedades de Texto

text-align: Define la alineación del texto. Ejemplo: text-align: center;

font-family: Define la fuente del texto. Ejemplo: font-family: Arial, sans-serif;

color: Establece el color del texto. Ejemplo: color: red;

2. Propiedades de Enlace

href: Establece el destino del enlace. Ejemplo: <a href="https://www.ejemplo.com">Enlace</a>

target: Define dónde se abre el enlace. Ejemplo: target="_blank" (abre el enlace en una nueva pestaña).

3. Propiedades de Imagen

src: Define la ubicación del archivo de imagen. Ejemplo: <img src="imagen.jpg">

alt: Proporciona un texto alternativo para la imagen. Ejemplo: <img alt="Descripción de la imagen">

4. Propiedades de Formulario

input type: Define el tipo de entrada en un formulario. Ejemplo: <input type="text">

label: Asocia un texto con un campo de formulario. Ejemplo: <label for="nombre">Nombre:</label>
